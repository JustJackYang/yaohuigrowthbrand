# 品牌咨询机器人 (Brand Consulting Bot)

这是一个基于“商业环境分析五大维度” (A企业、B品类、C竞争、D消费者、E渠道) 的50个问题构建的智能咨询助手。

## 功能特点

1.  **交互式对话**: 通过聊天界面逐步引导用户完成50个核心问题的思考与回答。
2.  **结构化分析**: 自动按五大维度整理用户的回答。
3.  **AI 智能诊断**: 支持接入 OpenAI (或兼容) API，根据用户的回答生成定制化的品牌诊断报告与战略建议。
4.  **通用建议**: 内置基于五维模型的通用咨询建议。

## 技术栈

- React
- Tailwind CSS
- Lucide React (图标)
- Framer Motion (动画)

## 如何运行

1.  安装依赖:
    ```bash
    npm install
    ```

2.  启动开发服务器:
    ```bash
    npm run dev
    ```

3.  在浏览器打开显示的地址 (通常是 http://localhost:5173)。

## 使用指南

1.  点击“开始咨询”进入问答流程。
2.  针对每个问题输入您的思考（支持跳过）。
3.  完成所有问题后，点击“生成报告”。
4.  在报告页，您可以查看整理好的回答。
5.  (可选) 点击“配置 AI”，输入您的 API Key (如 OpenAI key)，获取深度分析结果。

## 关于问题来源

问题源自营销大师叶明桂的《出圈》一书，涵盖企业、品类、竞争、消费者、渠道五个维度的50个灵魂拷问。
